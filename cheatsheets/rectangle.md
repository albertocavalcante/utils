# Rectangle Ultrawide Shortcuts

| Shortcut                 | Action          | Description                                        |
| :----------------------- | :-------------- | :------------------------------------------------- |
| **`Ctrl + Opt + C`**     | **Center Half** | **Focus Mode** - Perfect for your main IDE/Browser |
|                          |                 |                                                    |
| `Ctrl + Opt + D`         | First Third     | Left Sidebar (e.g. File Tree, Slack)               |
| `Ctrl + Opt + F`         | Center Third    | Narrower Focus                                     |
| `Ctrl + Opt + G`         | Last Third      | Right Sidebar (e.g. Docs, Terminal)                |
|                          |                 |                                                    |
| `Ctrl + Opt + E`         | First 2/3       | Wide Code View (Left aligned)                      |
| `Ctrl + Opt + T`         | Last 2/3        | Wide View (Right aligned)                          |
|                          |                 |                                                    |
| `Ctrl + Opt + Enter`     | Maximize        | Full Screen                                        |
| `Ctrl + Opt + Backspace` | Restore         | Undo Snap                                          |

## Tips

- Use **Center Half** (`^‚å•C`) as your default state.
- Use **Thirds** (`D`, `F`, `G`) when multitasking with 3 windows.
